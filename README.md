# Bob-Muse
适配 V2Board /v1.7.4 版本,其他版本自测

# 介绍
本主题由@geekui分享  
分享人TG：https://t.me/geekui  
TG频道：https://t.me/V2boradUI  

# 安装
下载到theme目录过后要修改当前文件夹名为Muse，然后在后台选择主题 【Bob-Muse】 就行了
